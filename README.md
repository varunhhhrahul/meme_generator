## To run the app

- Clone the repo.

- Open the project folder.

- In the terminal, run:

```dart
flutter pub get
```

- Run the app:

```dart
flutter run
```

- Select an Android or Ios device and you are good to go.

## Features

- Generate memes from a list of existing templates.

- Edit existing templates for memes.

- Save memes to the gallery.

- Add new templates.
